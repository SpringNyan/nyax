安装

Nyax 需要 TypeScript 的版本不低于 3.7。

Nyax 依赖于 Redux，Redux Observable，RxJS 和 Immer。

npm install nyax redux redux-observable rxjs immer

如果你希望和 React 进行集成，推荐使用 React Redux。

npm install react-redux
